<p>
  Building it:
  
  <ol>
    <li>Clone the repo with <code>git clone https://github.com/TheClashFruit/Rithle.git</code></li>
    <li>Go into the folder with <code>cd Rithle</code></li>
    <li>Build Splum with <code>./gradlew build</code></li>
  </ol>
  
  That's it!
</p>
